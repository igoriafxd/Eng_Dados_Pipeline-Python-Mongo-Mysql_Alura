# Engenharia de Dados: Pipeline Python-Mongo-MySQL

Bem-vindo ao repositório do projeto de Engenharia de Dados! Este projeto foi desenvolvido como parte do curso de Engenharia de Dados, onde você aprenderá a construir um pipeline de dados com Python, utilizando MongoDB e MySQL.

## Sobre o Curso

- Como construir um pipeline de dados com Python
- Configuração e utilização do MongoDB Atlas com a biblioteca Pymongo
- Aplicação de transformações nos dados
- Instalação e configuração do MySQL no WSL (Subsistema do Windows para Linux)
- Conexão do MySQL com Python
- Estruturação de códigos Python em funções

## Pré-requisitos

Antes de começar, certifique-se de ter instalado:

- Python 3.x
- MongoDB Atlas
- MySQL no WSL (se estiver utilizando Windows)
- Bibliotecas Python necessárias (verifique o arquivo `requirements.txt`)

## Como Utilizar

1. Clone este repositório:
  git clone https://github.com/igoriafxd/Eng_Dados_Pipeline-Python-Mongo-MySQL_Alura.git

2. Instale as dependências:
  pip install -r requirements.txt


3. Execute os scripts conforme necessário, seguindo a estrutura do projeto.

## Estrutura do Projeto

O projeto está organizado da seguinte forma:

- `data/`: Pasta para armazenar dados de entrada e saída.
- `scripts/`: Scripts Python para executar tarefas específicas.
- `README.md`: Este arquivo README.
